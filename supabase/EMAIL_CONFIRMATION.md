# Подтверждение email (защита от спама)

1. В Supabase: **Authentication → Providers → Email** включите **Confirm email**.
2. **Authentication → URL configuration**: укажите **Site URL** (например `http://localhost:5173`) и **Redirect URLs** для ссылок из письма.
3. В шаблоне письма пользователь переходит по ссылке и только после этого может войти.

Без подтверждения аккаунты с одноразовых почт смогут регистрироваться без ограничений — включение подтверждения решает это на стороне Supabase.
